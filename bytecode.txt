Defines:
defvar - Define Variable, Evaluates it
defnum - Define Number
defnul - Define Null
defbol - Define Bool
defstr - Define String
deftyp - Define Type
defety - Define Empty
deffun - Define Function/Closure
deflst - Define List
defdic - Define Dictionary

Control Flow:
jmpift - Jump if the top of the stack is true
jmpiff - Jump if the top of the stack is false
jmpict - Jump if the top of the stack is true, consume the top
jmpicf - Jump if the top of the stack is false, consume the top
jmprel - Force jump
bloent - Enter block
bloext - Exit block
blocrt - Create block

Dos:
docall - Call functions
doasgn - Assignment
dofind - Access members
doretn - Return expression
dooinc - Postfix increment
doodec - Postfix decrement
dobrak - Ghost break
docont - Ghost continue

Stack:
stkpop - Pop stack
